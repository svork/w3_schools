Web Storage no HTML5

Este recurso visa substituir os antigos cookies, que eram utilizados para armazenar informações

Há dois tipos principais:

  • localStorage: Armazena dados sem tempo de expiração
  • sessionStorage: Os dados são perdidos quando a aba do navegador é fechada
